# 我的私密笔记

这是一篇私密文章示例。访问这篇文章需要通过 Basic Auth 认证。

## 认证信息

默认用户名: `admin`
默认密码: `password123`

> **注意**: 部署后请在 Cloudflare Dashboard 的环境变量中设置 `BASIC_AUTH_USER` 和 `BASIC_AUTH_PASS` 来修改默认密码。

## 如何访问私密文章

1. 访问 `/private/private-notes/` 路径
2. 浏览器会弹出认证对话框
3. 输入用户名和密码
4. 认证成功后即可查看内容

## 使用场景

- 个人学习笔记
- 项目内部文档
- 敏感信息记录
- 草稿和未发布内容
